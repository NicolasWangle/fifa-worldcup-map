## 图片规格建议

- 尺寸：800x450 像素（16:9比例）
- 格式：JPG、PNG 或 WebP
- 大小：建议不超过 500KB

## 工作原理

- 系统会优先尝试加载本地图片
- 如果图片不存在或加载失败，会自动显示漂亮的渐变背景
- 每个场馆都有独特的配色方案
- 点击图片可以查看大图
